Project Name: Premium Kundali Report
====================================

Overview
--------
The Premium Kundali Report is a modern, responsive React-based web application that generates and displays comprehensive Vedic astrology (Kundali) reports. It provides users with deep insights into their cosmic alignments based on their birth details. The application features a beautiful, book-like interface with animations and rich visual elements.

Technology Stack
----------------
- Frontend Framework: React 19
- Build Tool: Vite
- Routing: React Router DOM v7
- Styling: Tailwind CSS (v4)
- Animations: Motion (Framer Motion)
- Icons: Lucide React
- Language: TypeScript

Core Features
-------------
1. Interactive Birth Details Form: Collects user information such as name, birth date, time, location (country/city), gender, and language preference.
2. Animated Loading Screen: Displays cosmic-themed loading animations while "calculating" the astrology report.
3. Multi-Page Report Interface: The generated report is presented in an interactive, book-like format with multiple pages, including:
   - Welcome & Summary
   - Birth Star (Nakshatra)
   - Core Personality Traits
   - Big Three Signs (Sun, Moon, Ascendant)
   - Five Great Elements Analysis (Fire, Earth, Air, Water, Ether)
   - Lagna Chart
   - Karmic Chakra & Stored Karma
   - Planetary Strengths & Profiles
   - Atmakaraka (Soul Planet)
   - Dasha Timeline (Planetary Periods)
   - Premium Deliverables Summary

Setup and Installation
----------------------
Prerequisites: Node.js

1. Clone the repository and navigate to the project directory:
   cd premium_kundali_report

2. Install the necessary dependencies:
   npm install

3. Set up environment variables:
   Ensure you configure the `GEMINI_API_KEY` in the `.env.local` file.

4. Start the development server:
   npm run dev

5. Build for production:
   npm run build
   This will output the optimized build to the `dist` directory.

Data Models (types.ts)
----------------------
The project uses strict TypeScript interfaces to maintain data integrity. Key interfaces include:
- BirthDetails: Captures user input.
- KundliReportData: The comprehensive object containing all generated astrology data (personality, elements, chakras, doshas, etc.).
- PlanetPosition, DashaPeriod, ChakraDetails: Sub-models for specific astrological aspects.
